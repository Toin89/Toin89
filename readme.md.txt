# Currículo Generator

Este projeto é um gerador de currículos online utilizando PHP, HTML, CSS, JavaScript e Bootstrap.

## Funcionalidades

- Formulário para entrada de dados pessoais e experiências profissionais.
- Cálculo automático da idade com JavaScript.
- Adição dinâmica de campos para experiências profissionais.
- Geração de currículo em formato imprimível.

## Como usar

1. Clone o repositório:
    ```
    git clone https://github.com/seu-usuario/curriculo-generator.git
    ```
2. Navegue até o diretório do projeto:
    ```
    cd curriculo-generator
    ```
3. Abra o `index.php` em um servidor local.

## Estrutura do Projeto

- `css/`: Contém os arquivos CSS.
- `js/`: Contém os arquivos JavaScript.
- `templates/`: Contém os templates HTML para gerar o currículo.
- `index.php`: Página principal com o formulário.
- `generate.php`: Script para gerar o currículo.
- `README.md`: Este arquivo.

## Tecnologias Utilizadas

- PHP
- HTML
- CSS (Bootstrap)
- JavaScript (jQuery)
